Problem & Solution

The Problem

기존 국방 로봇 도입은 특정 하드웨어 제조사에 종속(Vendor Lock-in)되어 유연한 편제가 불가능합니다. 또한, Nondeterministic한 작전 환경에서 폐쇄망(M-BcN) 보안을 충족하는 통합 관제 솔루션이 부재합니다.

Our Solution

P-Fleet은 VDA 5050 국제 표준을 활용한 하드웨어 중립적(Hardware-agnostic) 플랫폼입니다. Fleet-Level AI를 통해 Adaptive agent allocation을 수행하며, 군 보안 규격을 충족하는 안전한 Orchestration layer를 제공합니다.

Core Architecture

Orchestration Layer

단순한 경로 제어를 넘어선 Agentic AI 플랫폼입니다. 다중 로봇(Multi-agent) 간의 Context handoffs와 Task progress를 관리하며, ROS2/MQTT 통신을 통한 실시간 Workflow control을 수행합니다.

Visibility & Observability

작전 중 발생하는 모든 Reasoning-path traceability를 확보합니다. 강화학습 모델의 의사결정 과정과 로봇의 상태 데이터를 시각화하여 지휘관에게 Real-time dashboard를 제공합니다.

Governance & Security

군납의 핵심인 보안성을 설계 단계부터 내재화(Security by design)했습니다. M-BcN 연동 안정성, HMAC 서명을 통한 명령 무결성 검증, 그리고 국정원 CC 인증 기준을 준수하는 Data access control을 보장합니다.

Business Model

SaaS License

로봇 단말기(Endpoint) 1대당 과금하는 구독형 모델을 적용합니다. 국방 특수성에 맞춘 프리미엄 단가 책정으로 초기부터 안정적인 Cash flow를 확보합니다.

Predictive Maintenance

실시간 Telemetry 데이터를 분석하여 고장을 사전에 예측합니다. 가동률 극대화를 위한 부가적인 Insight 제공으로 LTV(Life Time Value)를 높입니다.

MRO & Consulting

기존 제조사들이 VDA 5050 표준을 준수할 수 있도록 기술 컨설팅 용역을 제공하고, 지속적인 System Integration 및 유지보수(MRO) 수익을 창출합니다.
